유튜브 플레이리스트 생성기

추출된 링크 (0개)

💡 데모 버전입니다. 실제 플레이리스트를 만들려면 위의 "Google로 로그인" 버튼을 클릭하세요.

📋 사용 방법

  1. 유튜브 링크가 포함된 텍스트를 위 입력창에 붙여넣기
  2. 자동으로 추출된 링크 확인
  3. 플레이리스트 이름 입력
  4. Google 계정으로 로그인 (선택사항)
  5. "플레이리스트 생성하기" 버튼 클릭
⚠️ 중요: 실제로 작동하려면 Google Cloud Console에서 YouTube Data API v3를 활성화하고 OAuth 2.0 클라이언트 ID를 발급받아야 합니다.
